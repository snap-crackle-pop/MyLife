B — Inline Actions on Each Row

Folder list is the main view. Each folder row has inline rename/delete icon buttons. Clicking a folder highlights it and shows its notes in the right panel. No separate panel slides in — actions live on the row.

C — Context Menu (right-click / ⋯)

Folder list in sidebar stays minimal. A ⋯ button or right-click on any folder opens a small context menu with Rename and Delete. Folder contents appear in the right panel on click. Clean, less visual noise.
